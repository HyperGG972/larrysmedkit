# larrysmedkit
Larry's Med kit is an open-source first aid kit for computers. It is a compact operating system that runs on a bootable USB, and it allows for file recovery, system error diagnosis, unmounting and partitioning of the disk in partitions which are currently active, and some basic kernel-level automation scripts


In its current state, Larry's Med Kit is still in bare metal form, having only the necessary code to run a blank screen while uniting hardware in bare metal form
